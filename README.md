Airport Infrastructure Analysis (SQL)
Project Overview

This project analyzes airport infrastructure data using SQL.
The objective is to explore patterns in airport activity such as passenger traffic, runway length, and IFR operations.

The dataset used in this project is synthetic and was created to practice SQL queries and basic data analysis.

Dataset

The dataset contains information about international airports including:

- airport name

- city

- country

- runway length

- annual passenger traffic

- number of terminals

- average daily flights

- IFR operation percentage

- airport altitude

Questions Explored

The analysis focuses on answering questions such as:

- Which airports have the highest passenger traffic?

- Which countries have the highest total passenger traffic?

- Which airports have the highest percentage of IFR operations?

- Do airports with longer runways tend to handle more passengers?

Tools Used

- SQL

- MySQL

Key SQL Concepts Used =

- SELECT

- WHERE

- ORDER BY

- GROUP BY

- HAVING

- SUM

- AVG

- COUNT
